BoNUS

Bonus for your university life, better(B) organize(o) your NUS Life.

  • This is a desktop personal organizer application. It is oriented for typist user (most operations should be done using keyboard rather than mouse).

  • It is a Java application intended for students learning Software Engineering in CS2103T at NUS.

  • The target user group is NUS students. This application aims at making their university life more convenient.

  • Users can manage and store information about their contacts, upcoming events and todo list.
